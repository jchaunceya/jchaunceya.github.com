Tweet Classification and Named Entity Extraction
assisting in educational research with machine learning tools

Lead a team of students creating a Python classification program for twitter data to assist professor in his research. We manually annotated tweets and stored them in separate MySQL database, set up data cleansing / feature vector regularization pipeline for efficient processing. We looked at several different classifiers: Multinomial/Bernoulli Naive Bayes, Decision Tree, Neural Networks, Random Forest, but ended up selecting support vector machine classifier algorithm and tweaked parameters as well as testing different data cleansing techniques to improve accuracy.
