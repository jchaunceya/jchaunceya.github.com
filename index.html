projects

Twitter Data processing and Classification and Named Entity Extraction
assisting in educational research with machine learning tools

In recent years, many universities have announced new campaigns focused on improving access to higher education to students from traditionally marginalized communities.

Once such strategy is to hire regional recruiters to speak to high schoolers, either individually or as a presentation to a group of students.

Often, these recruiters advertise their visits through Twitter.

We were interested in obtaining information about these events.

Lead a team of students creating a Python classification program for twitter data to assist professor in his research. We manually annotated tweets and stored them in separate MySQL database, set up data cleansing / feature vector regularization pipeline for efficient processing. We looked at several different classifiers: Multinomial/Bernoulli Naive Bayes, Decision Tree, Neural Networks, Random Forest, but ended up selecting support vector machine classifier algorithm and tweaked parameters as well as testing different data cleansing techniques to improve accuracy. Identify tweets relevant to research with program
